- (BOOL) open: (NSString *)  path

매개변수
pathNCG 파일의 경로
isPrepare복호화 가능상태로 파일을 오픈할 것인지의 여부.

반환값
성공일 경우 YES, 실패는 NO 리턴한다.

◆ seek:method:error:

- (BOOL) seek: (int64_t)  offset
method: (SeekMethod)  method
error: (NSError **)  error 

읽어들일 파일포인터의 위치를 이동시킨다.

매개변수
offset파일포인터의 위치를 이동시킬 offset를 해당 파라미터로 지정한다.
methodSeekMethod enum값 중 하나가 될 수 있다.
Begin 은 0인덱스기반으로 파일포인터의 offset를 지정하는 것이고
Current 는 현재의 파일포인터의 위치를 기반으로 offset을 지정하는 것이며
End 는 끝 위치로 부터의 offset을 지정하는 것이다.
표준 C 함수의 fseek의 seek method사용방법과 동일하게 사용된다.
error오류가 발생했을 경우 오류 정보를 자세히 확인하기 위해 error 객체를 전달할 수 있다.
nil로 전달한 경우에는 오류가 발생했을 경우 자세한 내용을 알 수 없다.
반환값
성공 시 YES, 실패 시 NO 리턴한다.
