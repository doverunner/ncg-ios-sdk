◆ initialize:rodbPath:deviceId:error:

- (BOOL) initialize: (OfflineSupportPolicy)  policy
rodbPath: (NSString *)  rodbPath
deviceId: (NSString *)  deviceId
error: (NSError **)  error 

NCG library has to be initialized by calling the relevant method before using it.


Secure Time value is verified internally while initialization.
If the device is on-line, Secure Time is acquired through on-line time server.
Device time value is used if it is not connected with Internet service.

Parameters
policyA policy information that how to process when the device is off-line.
rodbPaththe path of RoDB(license DB) which will be used inside of library. If the value is nil, it is created in the sandbox of the application.
deviceIdthe DeviceID which will be used in library. If the value is nil, It is generated automatically by SDK.
errorcheck error if nil no. succeeded if nil.
Returns
YES if succeeded. NO if failed.

It requests the license information of Token to license sever.


The method extracts the necessary information from the base64 encoded Token and requests the license.
The Token contains the Content ID, Site ID, and User ID of the NCG header information,
so that it is not necessary to download the NCG file in order to acquire the license.
For example, when acquiring license for HLS, PD contents, network resource and time can be saved.

◆ sendCustomRequest:siteID:requestData:error:

- (NSString *) sendCustomRequest: (NSString *)  acquisitionUrl
siteID: (NSString *)  siteID
requestData: (NSString *)  requestData
error: (NSError **)  error 

Send a custom request to NCG License-Server.

NCG Licese-Server can be comunicated with Client SDK by PKI Secure-Channel. This method provides some kind of Custom PKI Secure-Channel. This method sends a custom request to the License-Server and then the License-Server redirect the request to gateway-page to process the request. The response data that gateway-page responded is returned to license-server and then license-server respond the data to Client SDK.

Parameters
acquisitionUrlLicenseServer URL
siteIDsiteID
requestDatarequest data
error실패한 경우 해당 error객체를 통해 자세한 정보를 확인할 수 있다
Returns
Response string that the license-server responded
